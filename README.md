# Electron プロンプトテンプレートエディタ

変数を含むプロンプトテンプレートを作成・編集し、リアルタイムでプレビューできるElectron製デスクトップアプリケーションです。

## 主な機能

-   **2カラムレイアウト**: 左側でテンプレートを編集し、右側でリアルタイムにプレビューを確認できます。
-   **動的な変数検出**: `{変数名}` 形式の変数を自動的に検出します。
-   **インタラクティブな変数フォーム**: 検出された変数に対応する入力フォームを動的に生成します。
-   **リアルタイムプレビュー**: テンプレートや変数の値が変更されると、即座にプレビューが更新されます。
-   **エスケープ文字のサポート**: `\{` や `\}` を使うことで、波括弧を通常の文字として扱うことができます。

## 開発セットアップ

ローカル環境で開発を行うための手順は以下の通りです。

### 前提条件

-   [Node.js](https://nodejs.org/) (npmを含みます) がインストールされていること。

### インストールと起動

1.  **リポジトリをクローンします:**
    ```bash
    git clone https://github.com/hirokita117/prompt-editor.git
    cd prompt-editor
    ```

2.  **依存関係をインストールします:**
    ElectronやTailwind CSSなど、開発に必要なパッケージをインストールします。
    ```bash
    npm install
    ```

3.  **CSSをビルドします:**
    このアプリケーションはTailwind CSSを使用しています。以下のコマンドで `styles.css` から `output.css` を生成する必要があります。

    開発中は、別のターミナルで以下のコマンドを実行し、ファイルの変更を監視すると便利です。HTMLやJSファイルのクラス名が変更されるたびに、自動でCSSが再ビルドされます。
    ```bash
    npm run build:css
    ```

    一度だけビルドする場合は、以下のコマンドを実行します。
    ```bash
    npm run build:css -- --watch=false
    ```


4.  **アプリケーションを起動します:**
    依存関係のインストールとCSSのビルドが完了したら、以下のコマンドでElectronアプリケーションを起動します。（CSSの監視プロセスを実行している場合は、別のターミナルで実行してください）
    ```bash
    npm start
    ```

これでアプリケーションウィンドウが開き、プロンプトエディタを使用できるようになります。
